# 刘正要律师个人网站

域名：`liuzhengyaolawyer.com`

推荐部署方式：

1. 将本目录上传到 GitHub 仓库。
2. 在 Cloudflare Pages 中连接该 GitHub 仓库。
3. Framework preset 选择 `None`。
4. Build command 留空。
5. Build output directory 填 `/`。
6. 部署成功后，在 Cloudflare Pages 的 Custom domains 中添加：
   - `liuzhengyaolawyer.com`
   - `www.liuzhengyaolawyer.com`

## 后续更新

- 首页内容：编辑 `index.html`
- 样式：编辑 `styles.css`
- 文章：在 `articles/` 中新增 HTML 文件，并把链接加入首页和 `sitemap.xml`
- 图片：放入 `assets/`
